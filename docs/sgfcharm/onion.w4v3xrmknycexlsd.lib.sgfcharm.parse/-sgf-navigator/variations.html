variations

@Api fun variations(successors: Boolean = true): List<Move?> (source)

Returns the currently possible variations.

If successors is set to true (default), the variations of the successor node are returned, otherwise, the variations of the current node. The returned value is a list of [SgfType.Move]?s, each of which represents a variation in the order in which they appear in the tree. If the variation does not contain a move, the corresponding entry is null.
