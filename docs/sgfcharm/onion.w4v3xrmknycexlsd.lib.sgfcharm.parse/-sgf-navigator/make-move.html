makeMove

fun makeMove(move: SgfProperty<Move>, variationNumber: Int? = null): SgfNode? (source)

Carries out the move in the tree.

There are three possible situations:

  • the next node in the tree has the move: move to that node
  • we are at the last node of the current sequence and there are variation trees, one of which contains the move: move to the tree containing it
  • neither of the above: the user is branching off, a new variation is created

If variationNumber is not null, the navigator is forced to move to the variation with that index if it exists. This is useful for letting the user navigate to variations without moves in their root nodes, for example by displaying markup at custom places on the board.
