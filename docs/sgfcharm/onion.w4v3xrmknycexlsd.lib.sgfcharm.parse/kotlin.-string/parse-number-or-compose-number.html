parseNumberOrComposeNumber

@Impl fun String.parseNumberOrComposeNumber(): Compose<Number, Number>? (source)

Parses this string given either as a single sgf Number or a composed Number : Number.

In the former case, the returned object is Compose with both components identical to the Number. This is for parsing the SZ property which determines the board size. As it might be subject to change, I do not recommend using it directly.
