GoCoordinateParser

object GoCoordinateParser : CoordinateParser<XYPoint> (source)

A coordinate parser for the Go game.

The Point, Move and Stone data types are all the same in Go. A Point consists of two letters, 'a'..'z' + 'A'..'Z', where the first letter represents the column and the second one the row (see SgfType.Point for details).
