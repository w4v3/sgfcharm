NodeInfo

data class NodeInfo : SgfData (source)

A piece of information relating to the current node to be displayed in the text window.

Anything can be used for the key as long as the SgfViewAdapter knows how to deal with it, but by default, the SgfNodeHandler uses the values from SgfInfoKeys to encode which SGF property this info came from.
