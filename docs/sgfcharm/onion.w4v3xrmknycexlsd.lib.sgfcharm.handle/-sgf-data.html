SgfData

sealed class SgfData (source)

Base class for all instructions understandable by SgfView.

This is to cast the semantics of the sgf format into the semantics of what actually needs to be drawn in the end, which might be different. For example, the SgfView always needs a complete representation of the current board configuration, but each sgf node contains only incremental changes to the board.
