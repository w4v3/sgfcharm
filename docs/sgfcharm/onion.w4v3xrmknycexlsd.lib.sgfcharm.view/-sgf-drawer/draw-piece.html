@Beta abstract fun Canvas.drawPiece(piece: Piece, x: Float, y: Float, size: Float, paint: Paint): Boolean (source)

Draws the piece at screen coordinates x and y, with the given size and paint, returning true if the piece was drawn.

The size parameter is the distance between two lines of the board.
