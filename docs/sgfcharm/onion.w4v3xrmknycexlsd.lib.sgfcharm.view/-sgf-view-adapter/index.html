SgfViewAdapter

@Beta interface SgfViewAdapter (source)

If you want to use the SGF parsing and handling architecture provided by this library but make your own view with fundamental changes, you can make it implement this interface so that it can be controlled by the SgfController.

If you only need to make adjustments to the way some components are drawn, consider customizing the provided SgfView. Support for different games can also be achieved by implementing the SgfBoard interface.

A cleaner architecture would be to have a different class than the view implement this adapter, and then have the adapter tell the view what to draw. For ease of standard usage, in this library, the SgfView implements this interface directly.

You can also reuse the GoBoard, GoPieceDrawer, DefaultMarkupDrawer and DefaultInfoTextMaker components used by the SgfView implementation for your own view.
