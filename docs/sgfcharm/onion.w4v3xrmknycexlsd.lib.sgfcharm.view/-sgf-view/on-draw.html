onDraw

fun onDraw(canvas: Canvas?): Unit (source)

Draws the view.

First, it draws the grid representing the board, then the Piece and Markup data using the supplied (or default) drawing components, then undo and redo buttons, and finally, the text obtained from the NodeInfo data using the SgfInfoTextMaker.
