usePieceDrawer

fun usePieceDrawer(gameId: Int, pieceDrawer: SgfPieceDrawer): Unit (source)

Makes the SgfView use the given pieceDrawer to handle the drawing of the pieces specific to the game of type gameId.

If no SgfPieceDrawer implementation is found for this game, the GoPieceDrawer is used.
