DefaultInfoTextMaker

val DefaultInfoTextMaker: SgfInfoTextMaker (source)

Simple default implementation of SgfInfoTextMaker to turn NodeInfos into a SpannedString.

It prints some information about the last move at the top and then each NodeInfo in the given order with the NodeInfo.key in bold, putting those that do not have a key (like comments by default) to the bottom.
