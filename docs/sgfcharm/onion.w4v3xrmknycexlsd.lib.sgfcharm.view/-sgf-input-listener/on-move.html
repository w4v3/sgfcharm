onMove

@Impl abstract fun onMove(move: Move, variationIndex: Int? = null): Unit (source)

Should be triggered when the user has input a move; transmits that move to the controller.

If the user has selected a variation that does not have an SgfType.Move, this should transmit the variationIndex of the selected variation. The move will then be ignored.
