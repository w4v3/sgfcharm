DefaultMarkupDrawer

val DefaultMarkupDrawer: SgfMarkupDrawer (source)

The default SgfMarkupDrawer used by the SgfView to draw the board markup.

You might want to reuse Canvas.drawArrow and Canvas.drawTriangle if you just want to change the paint with which they are drawn, and String.width/String.height for anything involving text. You can also delegate to the DefaultMarkupDrawer for markup types you do not want to change.
