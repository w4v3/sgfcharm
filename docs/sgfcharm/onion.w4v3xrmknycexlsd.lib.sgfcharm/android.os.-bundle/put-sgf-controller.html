putSgfController

fun Bundle.putSgfController(key: String?, controller: SgfController?): Unit (source)

Puts the current state of the controller into this Bundle with the key.

Note that user placed stones not in the SGF file will not be stored; instead, the last state from the SGF is put into the bundle.
