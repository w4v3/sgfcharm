SgfInfoKeys

object SgfInfoKeys (source)

This object holds string literals to be used as NodeInfo.keys.

They are used by the SgfNodeHandler and understood by the DefaultSgfDrawer. You might of course change their values to display node information differently in the view.

The naming convention used here is as follows: Every property turned into a NodeInfo has a corresponding key here with the name of that property as identifier. As an exception, the properties taking a SgfType.Color or SgfType.Double value as argument correspond to maps with an SgfType.Color.Value.BLACK and an SgfType.Color.Value.WHITE entry or SgfType.Double.Value.MUCH and SgfType.Double.Value.VERY_MUCH entry, respectively. This allows testing for a PL (player) property using if (key in PL.values), for example.

Another exception is the SgfProperty.AP property, which is included with two different strings for the application name (SgfInfoKeys.APN) and version (SgfInfoKeys.APV).
