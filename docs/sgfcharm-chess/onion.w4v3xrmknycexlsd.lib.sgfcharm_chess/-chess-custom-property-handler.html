ChessCustomPropertyHandler

@Beta val ChessCustomPropertyHandler: SgfCustomPropertyHandler (source)

Handles the custom chess properties.

The DEF[] property is used to set up the board with the default starting configuration for chess. The CHK[1/2] property is used to indicate a check (1) or checkmate (2) move in the current node.
